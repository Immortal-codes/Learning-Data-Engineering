
## ðŸ”¹ Flattening a Nested Dictionary (FAANG-use)

ðŸ”¥ Often needed in data engineering & JSON parsing.

```python
def flatten(d, parent_key='', sep='_'):
    items = {}
    for k, v in d.items():
        new_key = f"{parent_key}{sep}{k}" if parent_key else k
        if isinstance(v, dict):
            items.update(flatten(v, new_key, sep))
        else:
            items[new_key] = v
    return items

nested = {"a": {"b": {"c": 1}}, "d": 2}
print(flatten(nested))  # {'a_b_c': 1, 'd': 2}
```
